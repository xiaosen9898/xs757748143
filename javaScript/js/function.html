函数的意义 ​

TIP

  1. 概念：函数是一段可以重复使用的代码片段，可以传参，返回值；
  2. 作用：实现大量代码重复使用，维护方便

函数的参数 ​

TIP

  1. 形参：函数声明时的参数，相当于一个函数内部的局部变量；
  2. 实参：函数调用时的实际参数，相当于一个变量的值；
  3. 形参与实参不匹配问题：
  1. 参数可以有也可以没有；
  2. 形参与实参是一一对应的，每个参数用逗号隔开；
  3. 如果形参多于实参（即有变量没值），则结果 undefined；
  4. 如果实参多于形参（有值没变量），则取到形参的个数

 

函数的返回值 ​

TIP

  1. return：函数的最终结果通过 return 返回给函数的调用者；
  2. 注意：
  1. 如果没有 return，就返回默认值 undefined；
  2. 如果函数内部有 return,但是 return 后面没有接内容,那返回值也是 undefined；
  3. 如果函数内部有 return，则遇到第一个 return 后会直接结束函数

作用域和作用域链 ​

TIP

  1. 函数外部不能直接访问函数内部的成员；
  2. 函数内部可以访问函数外部的成员，如果函数内部找不到则往上一层作用域找；
  3. 方法的形参相当于方法的局部变量，所以在方法内部优先使用形参；
  4. 词法作用域：函数作用域是在函数定义的时候作用域就确定下来了，和函数在哪调用无关
js
var num1 = 10
var num2 = 20
function fn(num1) {
var num3
// 不管调用时是否传递了参数，都不能否认有一个形参，形参是方法的局部变量
num1 = 100 // 使用了形参，与全局的num1没有关系
num2 = 200 // 自己的作用域中没有num2,往外找全局作用域，使用了全局变量num2
num3 = 300 // 使用了自己作用域中的局部变量
console.log(num1) // 100
console.log(num2) // 200
console.log(num3) // 300
}
fn()
console.log(num1) // 10
console.log(num2) // 200
// 函数外部不能直接使用函数内部所声明的变量
// 1.访问对象不存在的属性或者访问数组不存在的索引位置---undefined
// 2.访问未定义的变量 -- 报错
console.log(num3) // 报错： num3 is not defined

arguments ​

TIP

  1. 概念：是一个伪数组，有 length 属性和下标
  2. 作用：
  1. 用来存储传递的所有实参；
  2. arguments 伪数组的元素是实参的值；
  • 注意：
  1. 只能在函数内部使用；
  2. 没有真数组的一些方法；
  3. 箭头函数没有 arguments，使用 ...args 代替

 

闭包 ​

TIP

  1. 定义：闭包 = 一个函数内部的环境 + 环境中的函数；

  2. 形成：当内部函数访问了外部函数的变量的时候，就会形成闭包；

  3. 作用：保护私有变量不被修改

  4. 解决的问题

  1. 闭包内的数据不允许外界访问；
  2. 要解决的问题就是间接访问该数据
  1. 实现方式：

函数嵌套函数，内存函数访问外部函数的变量

  1. 闭包的定义和基本用法
  1. 基本用法：
js
function outer() {
 const name = "Alice";

 function inner() {
   console.log(name);
 }

 return inner;
}

const closureFunction = outer();
closureFunction(); // 输出: "Alice"

// outer 函数在执行时创建了一个局部变量 name
// inner 函数是 outer 的内部函数，它可以访问 outer 函数的变量
// 当 outer 函数返回 inner 函数后，closureFunction 保存了 inner 函数的引用
// 即使 outer 函数已经执行完毕，inner 依然可以访问并打印 name 的值。这就是闭包

  1. 模拟私有方法和属性
js
function Person(name) {
 let age = 25; // 私有变量

 this.getName = function() {
   return name;
 };

 this.getAge = function() {
   return age;
 };

 this.setAge = function(newAge) {
   if (newAge > 0) {
     age = newAge;
   }
 };
}

const person = new Person("Alice");
console.log(person.getName()); // 输出: "Alice"
console.log(person.getAge());  // 输出: 25
person.setAge(30);
console.log(person.getAge());  // 输出: 30
