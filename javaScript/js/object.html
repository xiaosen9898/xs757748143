引用数据类型-对象 Object ​

对象的意义 ​

TIP

  1. 概念：对象是一个具体的事物，是一组相关属性（特征）和方法（行为）的集合
  2. 作用：和变量/数组一样,都是用来存放数据的
  3. 方法：在对象里面的叫方法，不需要声明，在外面的叫函数

对象操作属性 ​

TIP

  1. 访问属性：
  1. 使用 . ：obj.name，此方式本质上是一个语法糖；
  2. 使用 对象名['键'] ：obj['name']，当键是一个变量时，需要用此方式
  1. 区别：属性赋值给一个新的变量之后，点方式访问不到这个属性值了，而中括号方式可以
  2. 注意：
js
let a = 'name';
console.log(obj[a]);      // obj['name'] 效果等同于 obj.name
console.log(obj['a']);    // undefined    obj.a
                          // 如果属性已经是字符串了，那么就不用单引号了

对象的 API ​

TIP

Object API

API含义
Object.keys(obj)获取对象的属性名组成的数组
Object.values(obj)获取对象的值组成的数组
Object.entries(obj)获取对象属性名和属性值组成的数组
Object.fromEntries(entries)将属性名和属性值的数组转换为对象
Object.assign()将多个对象的属性混合到一起（后面覆盖前面）
Object.getPrototypeOf()获取一个对象的隐式原型
Object.setPrototypeOf()设置一个对象的隐式原型
Object.create()创建一个新对象，同时设置新对象的隐式原型

 

属性描述符 ​

TIP

  1. 背景：对于对象中的每个成员，JS 使用属性描述符来描述它们
js
const user = {
  name: 'Jack',
  age: 18
}

// 上面的对象，在JS内部被描述为：
{
  // 属性 name 的描述符
  name: {
    value: 'Jack',
    configurable: true,   // 该属性的描述符是否可以被重新定义
    enumerable: true,     // 该属性是否允许被遍历，会影响for-in循环
    writable: true        // 该属性是否允许被修改
  },
  // 属性 age 的描述符
  age: {
    value: '18',
    configurable: true,   // 该属性的描述符是否可以被重新定义
    enumerable: true,     // 该属性是否允许被遍历，会影响for-in循环
    writable: true        // 该属性是否允许被修改

  1. 两个特殊的配置：get() 和 set()

const obj = {};
Object.defineProperty(obj, 'age', {
  get(){        // 读取属性age时，得到的是该方法的返回值
    return 18;
  },
  set(val){     // 设置属性age时，会把值传入val，调用该方法
    console.log(val)
  }
})

console.log(obj.age);   // 输出：18
obj.age = 20;           // 输出：20
console.log(obj.age);   // 输出：18

冻结 ​

TIP

  1. Object.freeze(obj)：以冻结一个对象，该对象的所有属性均不可更改
js
const obj = {
  a: 1,
  b: {
    c: 3,
  },
};

Object.freeze(obj); //  冻结对象obj

obj.a = 2;      // 不报错，代码无效
obj.k = 4;      // 不报错，代码无效
delete obj.a;   // 不报错，代码无效
obj.b = 5;      // 不报错，代码无效

obj.b.c = 5;    // b对象没有被冻结，有效

console.log(obj); // {a:1, b:{ c: 5 } }

相同性判定 ​

TIP

  1. Object.is()：可以判断两个值是否相同
  2. 它和===的功能基本一致，区别在于：
  1. NaN 和 NaN 相等；
  2. +0 和-0 不相等
js
Object.is(1, 2);       // false
Object.is("1", 1);     // false
Object.is(NaN, NaN);   // true
Object.is(+0, -0);     // false
