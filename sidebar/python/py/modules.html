模块 ​

认识模块 ​

TIP

  1. 定义：一个 .py 文件就是一个模块

  2. 作用：模块可以被别的程序引入，以使用该模块中的函数等功能

 

模块导入 ​

TIP

1. import ​

  1. 语法：
  1. a_module.py：创建a模块
  2. import a_module：b模块导入a模块
  3. a_module.XXX：b模块使用a模块
  1. 作用：把某个模块的所有对象导入到另一个模块

 

2. from … import ​

  1. 语法：from moduleName import name1...

  2. 作用：把某个模块的一部分对象导入到另一个模块

 

3. from … import * ​

  1. 语法：from moduleName import *

  2. 作用：把一个模块的所有内容全都导入到当前的命名空间

 

__name__属性 ​

TIP

  • 每个模块都有一个__name__属性，当其值是'main'时，表明该模块自身在运行，否则是被引入

 

dir() 函数 ​

TIP

  • dir() 函数会罗列出当前定义的所有名称

os 模块 ​

TIP

os 模块提供了许多与操作系统交互的函数，例如创建、移动和删除文件和目录，以及访问环境变量等

 

sys 模块 ​

TIP

os sys 模块提供了与 Python 解释器和系统相关的功能，例如解释器的版本和路径，以及与 stdin、stdout 和 stderr 相关的信息

 

time 模块 ​

TIP

time 模块提供了处理时间的函数，例如获取当前时间、格式化日期和时间、计时等

 

datetime 模块 ​

TIP

  1. datetime 模块提供了更高级的日期和时间处理函数，例如处理时区、计算时间差、计算日期差等

  2. 方法：

  1. datetime.datetime.now()：获取当前日期和时间
  2. datetime.date.today()：获取当前日期
  3. current_datetime.strftime("%Y-%m-%d %H:%M:%S")：格式化日期

 

random 模块 ​

TIP

  1. random 模块提供了生成随机数的函数，例如生成随机整数、浮点数、序列等

  2. 方法：

  1. random.randint(n, m)：随机生成一个n-m的整数
  2. random.choice(['apple', 'pear', 'banana'])：从列表随机抽取一个数

 

math 模块 ​

TIP

math 模块提供了数学函数，例如三角函数、对数函数、指数函数、常数等

 

re 模块 ​

TIP

re 模块提供了正则表达式处理函数，可以用于文本搜索、替换、分割等

 

json 模块 ​

TIP

json 模块提供了 JSON 编码和解码函数，可以将 Python 对象转换为 JSON 格式，并从 JSON 格式中解析出 Python 对象

 

urllib 模块 ​

TIP

urllib 模块提供了访问网页和处理 URL 的功能，包括下载文件、发送 POST 请求、处理 cookies 等
