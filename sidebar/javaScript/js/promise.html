Promise ​

定义 ​

TIP

  1. 定义：
  1. Promise 是异步编程的一种解决方案；
  2. 语法上，Promise 是一个对象，从它可以获取异步操作的消息
  1. 作用：将异步操作以同步操作的流程表达出来

 

基本用法 ​

TIP

  1. 原理：Promise对象是一个构造函数，用来生成Promise实例
js
const promise = new Promise(function(resolve, reject) {
   setTimeout(()=>{
          // ... some code
      if (true){
        resolve(value);
      } else {
        reject(error);
      }
   }, 1000)
});

  1. 状态：
  1. Promise对象代表一个异步操作，有三种状态：pending（进行中）、fulfilled（已成功）、rejected（已失败）
  2. Promise对象的状态改变，只有两种可能：进行中→已成功、进行中→已失败
  1. 参数：
  1. 构造函数接收一个参数作为回调函数；
  2. 回调函数又接收两个参数作为回调函数：
  1. resolve回调函数：
  1. 将Promise对象的状态从“未完成”变为“成功”；
  2. 在异步操作成功时调用，将异步操作的结果，作为参数传递出去
  1. reject回调函数：
  1. 将Promise对象的状态从“未完成”变为“失败”；
  2. 在异步操作失败时调用，并将异步操作报出的错误，作为参数传递出去

 

promise的方法 ​

TIP

  1. then方法：
  1. 有两个参数，是两个回调函数：
  1. 第一个回调函数是Promise对象的状态变为resolved时调用，它会接收resolve(value)的参数，作为自己第一个参数的实参；
  2. 第二个回调函数是Promise对象的状态变为rejected时调用，它会接收reject(value)的参数，作为自己第一个参数的实参；另外，如果设置第二个回调参数，那么catch方法将不会执行
  1. 返回值：
  1. 返回：then方法返回的是一个新的promise实例，并且该promise的状态为pending
  2. 说明：promise的状态为pending时，不会执行then方法
  3. 改变：使用return可以可以将新的promise实例的状态改成fulfill，如果在then方法中，代码报错，那么新的promise实例的状态改成rejected
js
promise.then(
 (value) => {
   // 成功时的回调，value 是 resolve 函数传递的参数
   console.log(value);
 },
 (reason) => {
   // 失败时的回调，reason 是 reject 函数传递的参数
   console.log(reason);
 }
);

  1. catch方法：
  • 有一个参数，是一个回调函数：
  1. 回调函数是Promise对象的状态变为rejected时调用，它会接收reject(value)的参数，作为自己第一个参数的实参；
  2. 相当于是then()方法的第二个参数

 

  1. all()方法：
  • 作用：处理并发任务并确保所有任务都成功完成，在第一个失败时立即终止整个流程
js
// 示例1: 所有Promise都成功
Promise.all([
  Promise.resolve(1),
  Promise.resolve(2),
  Promise.resolve(3),
]).then(values => console.log(values)); // 输出: [1, 2, 3]

// 示例2: 有一个Promise失败
Promise.all([
  Promise.resolve(1),
  Promise.reject(new Error('Second promise failed')),
  Promise.resolve(3),
]).catch(error => console.error(error)); // 输出: Error: Second promise failed

  1. race()方法：
  • 场景：
  1. 适用于那些需要尽快响应或者需要有超时机制的场景；
  2. 处理多个异步时，谁先响应就返回谁的结果，无论成功与否
js
// 示例1: 第一个成功
Promise.race([
  new Promise(resolve => setTimeout(resolve, 1000, 'First')), // 1秒后解决
  new Promise(resolve => setTimeout(resolve, 500, 'Second')), // 0.5秒后解决
]).then(value => console.log(value)); // 输出: "Second"

// 示例2: 第一个失败
Promise.race([
  fetch('url1', { timeout: 500 }), // 500毫秒后超时
  fetch('url2'), // 正常请求，可能需要更长时间
]).catch(error => console.error(error)); // 如果url1超时，将输出错误信息

async和await ​

TIP

  1. 定义：promise的语法糖

  2. async：

  1. 作用：在异步函数中，简化创建Promise；
  2. 使用：将async关键字放在使用了await关键字所在函数的前面（被async修饰的函数返回的结果一定是一个Promise对象）
  1. await：
  1. 作用：等待异步函数的结果；
  2. 使用：放在异步代码的前面（必须与async搭配使用）
js
// 示例1
function test1() {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve('异步函数执行正常的结果')
        }, 3000)
    })
}

async function test2() {
    const res = await test1()
    console.log('res', res);  //res 异步函数执行正常的结果
}
test2()

//示例2      
function ask(person) {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            const random = Math.round(Math.random() * 10)
            console.log(`${person}开始执行，数字为：${random}`);
            if (random > 6) {
                resolve(true)
            } else {
                resolve(false)
            }
        }, 3000)
    })
}

async function test() {
    const list = ['东', '西', '南', '北']
    for (let i = 0; i < list.length; i++) {
        let per = list[i]
        const res = await ask(per)
        if (res) {
            console.log('已成功');
            break;
        } else {
            console.log('失败了');
        }
    }
}
test()
