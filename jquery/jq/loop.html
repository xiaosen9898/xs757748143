向上遍历 ​

TIP

  1. parent()：返回被选元素的直接父元素；

  2. parents()：

  • 返回被选元素的所有祖先元素，直到根元素
  • 可以使用可选参数来过滤祖先元素
js
/* 返回所有 <span> 元素的所有祖先，并且它是 <ul> 元素 */
$(document).ready(function(){
  $("span").parents("ul");
});

向下遍历 ​

TIP

  1. children()：
  • 返回被选元素的所有直接子元素
  • 可以使用可选参数来过滤对子元素
js
/* 返回类名为 "1" 的所有 <p> 元素，并且它们是 <div> 的直接子元素 */
$(document).ready(function(){
  $("div").children("p.1");
});
1

  1. find()：返回介于两个给定元素之间的所有祖先元素
  • 返回被选元素的后代元素，一路向下直到最后一个后代
  • 可以使用可选参数来过滤对子元素
js
/* 返回属于 <div> 后代的所有 <span> 元素 */
$(document).ready(function(){
  $("div").find("span");
});

/* 返回属于 <div> 后代的所有 <span> 元素 */
$(document).ready(function(){
  $("div").find("*");
});

同级元素 ​

TIP

  1. siblings()：
  • 返回被选元素的所有同胞元素
  • 可以使用可选参数来过滤同胞元素
js
/* 返回属于 <h2> 的同胞元素的所有 <p> 元素 */
$(document).ready(function(){
  $("h2").siblings("p");
});
1
2
3
4

  1. next()：返回被选元素的下一个同胞元素；

  2. nextAll()：返回被选元素的所有跟随的同胞元素；

  3. nextUntil()：

  • 返回介于两个给定参数之间的所有跟随的同胞元素
js
/* 返回介于 <h2> 与 <h6> 元素之间的所有同胞元素 */
$(document).ready(function(){
  $("h2").nextUntil("h6");
});
1
2
3
4

  1. prev()：返回被选元素的上一个同胞元素；

  2. prevAll()：返回被选元素的所有前面的同胞元素；

  3. prevUntil()：返回介于两个给定参数之间的所有前面的同胞元素

 

元素过滤 ​

TIP

  1. first()：
  • 返回被选元素的首个元素
js
/* 选取首个 <div> 元素内部的第一个 <p> 元素 */
$(document).ready(function(){
  $("div p").first();
});
1
2
3
4
  1. last()：
  • 返回被选元素的最后一个元素
js
/* 选择最后一个 <div> 元素中的最后一个 <p> 元素 */
$(document).ready(function(){
  $("div p").last();
});
1
2
3
4
  1. eq()：
  • 返回被选元素中带有指定索引号的元素，索引号从 0 开始，因此首个元素的索引号是 0 而不是 1
js
/* 选取第二个 <p> 元素（索引号 1） */
$(document).ready(function(){
  $("p").eq(1);
});
1
2
3
4
  1. filter()：
  • 不匹配这个标准的元素会被从集合中删除，匹配的元素会被返回
js
/* 返回带有类名 "url" 的所有 <p> 元素 */
$(document).ready(function(){
  $("p").filter(".url");
});
1
2
3
4
  1. not()：
  • 返回不匹配标准的所有元素
js
/* 返回不带有类名 "url" 的所有 <p> 元素 */
$(document).ready(function(){
  $("p").not(".url");
});
