Flex 布局 ​

TIP

Flex ​

给父盒子添加弹性布局属性：

css
 .father {
   display: flex;
 }
1
2
3
  1. 此时父盒子变为弹性盒子；
  2. 父盒子称为父项，子盒子称为子项

 

父项 ​

1.主轴：主轴为 X 时，默认自左向右，主轴为 Y 时，默认自上向下 ​

  1. 子项对齐方式（给父项设置）：
css
 .father {
   display: flex;
   display: flex-start;      /* 头对齐 （默认） */
   display: flex-end;        /* 尾对齐 */
   display: center;          /* 居中 */
   display: space-between;   /* 贴两边 */
   display: space-around;    /* 子项两边有相等距离 */
   display: space-evenly;    /* 空间平分 */
 }

2.侧轴：侧轴为 X 时，默认自左向右，侧轴为 Y 时，默认自上向下 ​

  1. 子项对齐方式（给父项设置）：

  2. 注意：

  1. 移动时，子项位置往轴的方向移动；
  2. 子项默认宽高情况看主轴就行

   display: flex;

   align-items: flex-end;        /* 下对齐 */
   align-items: center;          /* 居中 */

   align-content: flex-end;        /* 尾对齐 */
   align-content: center;          /* 居中 */
   align-content: space-between;   /* 贴两边 */
   align-content: space-around;    /* 子项两边有相等距离 */

子项 ​

  1. 设置子项伸缩比例：占父项剩余宽度的比例
css
 .child {
   flex: 1;      /* 沾满剩余宽度 */
 }

  1. 设置单个子项在侧轴上的对齐方式：
css

   align-self: flex-start;
   align-self: flex-end;
   align-self: center;
 }

Grid 布局 ​

TIP

Grid ​

作用：网格布局是多行多列布局的终极解决方案

css
 .father {
   display: grid;
   grid-template-rows: 200px 200px 200px;      /* 3行，每列宽度200px */
   grid-template-columns: 200px 200px 200px;

   /* 单元格之间的间隙 */
   row-gap: 10px;                              /* 行间隙为10px */
   column-gap: 20px;                           /* 列间隙为20px */
   gap: 10px 20px;                             /* 单元格行间隙为10px，列间隙为20px */
 }

  1. 改变排列方向：grid-auto-flow: row/column

 

  1. 单元格内部的对齐（默认情况下，网格项目在单元格内部水平和垂直拉伸，撑满单元格）
  1. 可以使用属性justify-items设置水平方向的排列方式
  2. 可以使用属性align-items设置垂直方向的排列方式
css
 .father {
   justify-items: start 左 | end 右 | center 中 | stretch 拉伸;
   align-items: start 上 | end 下 | center 中 | stretch 拉伸;

   /* 可以使用速写属性`place-items: 垂直对齐方式 水平对齐方式`同时设置这两个值 */
   place-items: start center; /* 垂直靠上，水平居中 */
 }

  1. 网格项目定位：grid-area
  1. 背景：默认情况下，网格项目依次排列到单元格中，每个网格占据一个单元格
  2. 作用：设置grid-area属性来改变这一行为
css
 .father {
   grid-area: 起始行线编号/起始列线编号/结束行线编号/结束列线编号;
 }
