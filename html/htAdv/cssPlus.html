CSS 变量 ​

TIP

  1. 概念：在 CSS 中，变量是用于存储和重复使用值的一种机制。CSS 变量也被称为自定义属性。

  2. 作用：提高代码的复用性，减少重复代码；

  3. 语法：

  1. 声明：--变量名: 值 → 使用：var(--变量名)
  2. 声明：--变量名: 值 → 使用：var(--变量名, 后备css属性)

注意：

  1. 后备 CSS 属性值只在前面的自定义属性一定无效的时候才渲染，如果 var()函数的第一个参数值可能有效，则后备 CSS 自定义属性值是不会渲染
  2. 如果第一个参数值是不合法的，则 var()函数解析为当前 CSS 属性的初始值或继承值（如果有继承性），也就是按照 unset 全局关键字的规则渲染
css
:root {
 --main-color: #3498db;
 --font-size: 16px;
}

  1. 变量命名：以--开头，限制较少，不支持$、[、]、^、(、)、%、"等字符，使用需要转义

  2. CSS 自定义属性值特点：

  1. CSS 自定义属性值可以是任意值或表达式
  2. CSS 自定义属性值可以相互传递
  1. 作用域：
  1. 全局作用域；
  2. 局部作用域；
  3. 继承，但与权重无关
  1. 行间样式也支持 css 变量

CSS calc() ​

TIP

  1. 作用：calc() 是 CSS 中的一个函数，用于在样式中进行动态计算，可以混合不同的单位进行运算，比如像素（px）、百分比（%）、em 等

  2. 语法：calc(表达式)

  3. 使用：

css
.box {
 width: calc(100% - 50px);
}

CSS attr() ​

TIP

  1. 作用：attr() 是 CSS 中的一个函数，用来获取 HTML 元素的属性值，并在样式中使用它

  2. 语法：content: attr(属性名);

  3. 使用：常见于 ::before 或 ::after 伪元素中

html
<a href="https://example.com">Example</a>

a::after {
  content: " (" attr(href) ")";
}

<!-- 效果：Example (https://example.com) -->

  1. attr() 现在只对 content 属性有效；
  2. 不适用于布局属性（如 width、color）——目前浏览器不支持；
  3. 如果要用 HTML 属性控制样式，一般通过 JS 或自定义属性（--var）+ var() 实现

 

CSS min() 和 max() ​

TIP

min(值 1, 值 2, ...) ​

  1. 作用：从多个值中选择最小的一个结果应用；

  2. 使用：属性: min(值1, 值2, ...);

css
 .box {
   width: min(100%, 500px);
 }
 /* 表示宽度最多为 500px，如果屏幕宽度小于 500px，就使用 100% */
1
2
3
4
css
 .container {
   width: min(90vw, 800px);
 }
 /* 表示容器宽度在 90% 视口宽度 和 800px 之间取较小的那个 */
1
2
3
4
css
 h1 {
   font-size: min(8vw, 40px);
 }
 /* 屏幕变小时，字体也会缩小，但最大不会超过 40px */
1
2
3
4
css
 .box {
   height: min(60vh, 400px);
 }
 /* 高度最大 60% 视口高，但不超过 400px */
1
2
3
4

⚠️ 注意

  1. 支持混合单位，如 px、%、em、vw、vh 等；
  2. 可与 max()、clamp() 等配合使用

 

max(值 1, 值 2, ...) ​

  1. 作用：从多个值中选择最大的一个结果应用，和 min() 是互补关系；

  2. 使用：属性: max(值1, 值2, ...);

css
 .container  {
   width: max(300px, 50%);
 }
 /* 容器宽度至少 300px，但在较大屏幕上可以是父元素的一半（50%） */
1
2
3
4
css
 body {
   font-size: max(16px, 1.5vw);
 }
 /* 保证字体至少 16px，但在大屏上更大 */
1
2
3
4
css
 .section {
   min-height: max(60vh, 500px);
 }
 /* 保证内容区高度至少 500px，在大屏幕上占 60% 视口高 */

4

⚠️ 注意

  1. 单位可以混合使用，如 px、%、vw、vh、em 等；
  2. 运算不需要空格：max(10px, 5vw)；

 

CSS clamp() ​

TIP

  1. 作用：用于在一个最小值和最大值之间指定一个理想值（推荐值），浏览器会根据环境（如视口大小）自动在这个范围内调整

  2. 语法：clamp(最小值, 理想值, 最大值);

  1. 最小值：不会比这个小
  2. 理想值：推荐值，会尽量使用它
  3. 最大值：不会比这个大
  1. 使用：
css
 h1 {
   font-size: clamp(20px, 5vw, 40px);
 }
 /* 字体大小最小 20px，最大 40px，理想是根据视口宽度的 5% */
1
2
3
4
css
 .container {
    width: clamp(300px, 80%, 1000px);
 }
 /* 宽度在 300px 和 1000px 之间，推荐为 80% 父容器宽度 */
1
2
3
4
css
 /* 用在 padding、margin、gap 等布局属性 */
 .section {
    padding: clamp(16px, 4vw, 48px);
 }
 /* 在不同屏幕下，自动调节内边距大小，既不太挤也不太空 */
