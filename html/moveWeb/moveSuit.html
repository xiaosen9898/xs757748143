了解屏幕 ​

TIP

1. 分辨率： ​

  1. 逻辑分辨率：屏幕的宽度和高度；
  2. 物理分辨率：屏幕中存放物理像素点个数，显示图像最小单位

2. PPI： ​

  1. 像素密度；
  2. 单位英寸内拥有物理像素点的个数

3. 视口： ​

  1. 早期 PC 端网页版心，大小 980px；
  2. 视口代码

 

移动端屏幕适配 ​

TIP

1. 屏幕适配： ​

根据设计稿开发网页，随着屏幕的变化，元素 size 也按比例变化

2. 主要技术： ​

  1. VW（主流）： ​
  1. vw 是相对长度单位；
  2. 100vw = 屏幕的宽度；
  3. 计算公式；
  1. 设计稿宽度/设计稿中元素宽度 = 手机屏幕宽度（100vw）/手机屏幕中元素的宽
  2. 手机屏幕中元素宽度 = 设计稿元素宽度/设计稿宽度 * 100vw
  1. 注意 ：使用插件变换为 vw 单位时， 首先要把插件中屏幕宽度改为设计稿的宽度；
  2. 开发流程；
  1. 先用 px 来完成全部的布局效果；
  2. 再将 px 转为 vw；

 

  1. rem + flexible.js： ​
  1. rem：
  1. rem 是相对长度单位；
  2. 大小是相对于页面根标签 html 的字体大小；
  3. 谷歌浏览器默认的字体大小为 16px
  1. flexible：
  1. 手机淘宝开发的 js 文件；
  2. 作用：检测屏幕大小，设置 html 的字体大小；
  3. 规定：10 rem = 屏幕的宽度；
  4. 计算公式
  1. 设计稿宽度/设计稿中元素宽度 = 手机屏幕宽度（10 rem）/手机屏幕中元素的宽度
  2. 手机屏幕中元素宽度 = 设计稿元素宽度/设计稿宽度 * 10 rem
  1. 开发流程：
  1. 首先引入 flexible.js 文件；
  2. 再用 px 来完成全部的布局效果；
  3. 最后再将 px 转为 rem

 

响应式布局 ​

1. 响应式 ​

TIP

  1. 概念：使用一套代码，在不同尺寸的屏幕下，让用户有良好的体验效果
  2. 底层原理：媒体查询：
  1. 概念：一种 css 技术，是屏幕适配的底层原理；
  2. 作用：根据屏幕宽度的不同，加载不同的 css 代码；
  3. 语法：
css
@media screen and  ( width: 800px ) { }      /* 当屏幕宽度等于800px时 */
@media screen and  ( max-width: 800px ) { }  /* 当屏幕宽度小于800px时 */
@media screen and  ( min-width: 800px ) { }  /* 当屏幕宽度大于800px时 */
@media screen and  ( min-width: 800px ) and (max-width:1000px) { }  /* 当屏幕大于800px又小于1000px时 */

2. Bootstrap ​

TIP

  1. 概念：一个响应式布局的 UI 框架；
  2. 作用：一套代码既可以用在 PC 端，也可以用在移动端；
  3. 核心：栅格系统：
  1. 规定：把屏幕分为 12 列；
  2. 四种宽度的屏幕：
  1. xs：极小屏幕，768px；
  2. sm：小屏幕，768px-992px；
  3. md：中等屏幕，992px-1200px；
  4. lg：大屏幕，大于 1200px；
  1. 使用栅格系统步骤：
  1. 定容器；
  1. .container： 版心
  2. ..container-fluid：通栏
  1. 定行：.row；
  2. 栅格代码：
  1. 看屏幕尺寸
  2. 一份占几列
  1. 注意：如果较大屏幕未设置，较大的屏幕会使用较小屏幕的设置
  2. 代码：
html

<!DOCTYPE html>
<html lang="en">
<head>
 <meta charset="UTF-8">
 <meta name="viewport" content="width=device-width, initial-scale=1.0">
 <title>Bootstrap Page</title>
<!-- 引入 Bootstrap （CDN 引入） -->
 <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/5.1.3/css/bootstrap.min.css"> 
</head>
<body>
 <!-- 页面内容 -->
<div class="container">
 <div class="row"> 
   <div class="col-sm-4">列1</div> 
   <div class="col-sm-4">列2</div> 
   <div class="col-sm-4">列3</div> 
 </div>
</div>

<!-- 引入 相关文件 -->

</body>
