HTML 标签 ​

语义化标签 ​

TIP

  1. 概念：每个 HTML 元素都有具体的含义

  2. 作用：

  1. 为了搜索引擎优化（SEO），搜索引擎会从整个互联网中抓取页面源代码
  2. 利于无障碍访问
  3. 利于浏览器的插件分析网页
  1. HTML 元素周期表
html
<h1>标题1标签</h1>
<p>段落标签</p>
...

2

  • 所有元素只是表示语义，与展示效果无关，展示效果应由 css 决定（浏览器带有默认的 css 样式，所以每个元素有一些默认的样式）

表格的其他标签：

  1. caption：表格整体的大标题，写在 table 标签开始标签的下面；
  2. th：表头标签（表格一列的小标题）；
  3. 表格的结构标签（用来包裹 tr）：
    1. 表格的头部：<thead></thead>
    2. 表格的身体：<tbody></tbody>
    3. 表格的底部：<tfoot></tfoot>
  4. 表格标签的属性：
    1. border：边框，默认是 0，一般设为 1；
    2. width/height：宽高；
    3. cellspacing：单元格之间的距离，设为 0 即为实线；
    4. cellpadding：内容与单元格之间的距离；
  5. 合并单元格：
    1. 先看拿掉行还是竖；
    2. 拿掉行：rowspan，拿掉竖：colspan；
    3. 有几个单元格合并属性值就是几，根据左上原则，保留最左或最上并删除其他涉及到的单元格
  6. 边框合并（给 table 标签设置）：border-collapse：collapse

label 标签 ​

作用：绑定文本和表单标签的关系

使用方式 1：

使用 for 属性，for 属性的值设置为表单元素的 id 值，这样点击 label 时会自动聚焦到对应的表单元素

html
<label for="username">用户名：</label>
<input type="text" id="username" name="username">

2

 

使用方式 2：
将 label 包裹表单元素，当 label 标签直接包裹表单元素时，无需设置 for 属性，点击 label 也能聚焦到对应元素

html
<label>用户名：
<input type="text" name="username">
</label>

媒体标签 ​

TIP

1.图片标签 ​

  1. 作用：在网页中显示一张图片
html
<img src="" alt="" title="" width="" height="">

1. 属性：
   1. src：图片的路径（必须要）
   2. alt：替换文本（只有当图片加载失败时，才会显示的文字）
   3. title：图片的标题（鼠标悬停在图片上时，才会显示的文字）
   4. width 宽度/height 高度：如果只设置了其中的一个，另一个会等比例缩放
   5. object-fit：图片的缩放方式（contain：不缩放；cover：填充整个容器；...）

注意：
   - 一般只会设置 width 或者 height 中的一个，如果同时设置了两个，图片可能会拉伸变形

链接标签 ​

TIP

作用：跳转

空链接：<a href="#">空链接</a>

html
<a href="路径"> </a>

属性：
   1. href：告诉浏览器点击之后去哪一个网页：
      1. 外部链接：直接写网址即可（网址需要写完整，https://）
      2. 本地网页：直接写路径
   2. target：设置跳转方式：
      1. target="\_self"：默认值，在当前窗口中进行跳转，原网页不在了
      2. target="\_blank"：在新窗口中进行跳转，原网页还在
