h 系列标签 ​

h1~h6，h1 为标题，h2 为副标题，h3 为三级标题，以此类推

显示特点：

  1. 文字加粗；
  2. 文字变大，从 h1 - h6 逐渐减小；
  3. 独占一行

<h2>2</h2>
<h3>3</h3>
<h4>4</h4>
<h5>5</h5>
<h6>6</h6>

表格标签 ​

显示的效果：格子表格的形式

html
<table>
 <thead>
   <tr> <!-- 表格头 -->
     <th>列1</th>
     <th>列2</th>
   </tr>
 </thead>
 <tbody>
   <tr> <!-- tr为一行 -->
     <td>数据1</td>  <!-- td为一行中的一格 -->
     <td>数据2</td>
   </tr>
   <tr>
     <td>数据4</td>
     <td>数据5</td>
   </tr>
 </tbody>
</table>

表格的其他标签：

  1. caption：表格整体的大标题，写在 table 标签开始标签的下面；
  2. th：表头标签（表格一列的小标题）；
  3. 表格的结构标签（用来包裹 tr）：
    1. 表格的头部：<thead></thead>
    2. 表格的身体：<tbody></tbody>
    3. 表格的底部：<tfoot></tfoot>
  4. 表格标签的属性：
    1. border：边框，默认是 0，一般设为 1；
    2. width/height：宽高；
    3. cellspacing：单元格之间的距离，设为 0 即为实线；
    4. cellpadding：内容与单元格之间的距离；
  5. 合并单元格：
    1. 先看拿掉行还是竖；
    2. 拿掉行：rowspan，拿掉竖：colspan；
    3. 有几个单元格合并属性值就是几，根据左上原则，保留最左或最上并删除其他涉及到的单元格
  6. 边框合并（给 table 标签设置）：border-collapse：collapse

9. label 标签 ​

作用：绑定文本和表单标签的关系

使用方式 1：

使用 for 属性，for 属性的值设置为表单元素的 id 值，这样点击 label 时会自动聚焦到对应的表单元素

html
<label for="username">用户名：</label>
<input type="text" id="username" name="username">
1
2

 

使用方式 2：
将 label 包裹表单元素，当 label 标签直接包裹表单元素时，无需设置 for 属性，点击 label 也能聚焦到对应元素

html
<label>用户名：
 <input type="text" name="username">
</label>

媒体标签 ​

TIP

1.图片标签 ​

作用：在网页中显示一张图片

html
<img src="" alt="" title="" width="" height="">

1. 属性：   
   1. src：图片的路径（必须要）
   2. alt：替换文本（只有当图片加载失败时，才会显示的文字）
   3. title：图片的标题（鼠标悬停在图片上时，才会显示的文字）
   4. width 宽度/height 高度：如果只设置了其中的一个，另一个会等比例缩放

注意：   
   1. 一般只会设置 width 或者 height 中的一个，如果同时设置了两个，图片可能会拉伸变形

链接标签 ​

TIP

作用：跳转

空链接：<a href="#">空链接</a>

html
<a href="路径"> </a>

属性：
   1. href：告诉浏览器点击之后去哪一个网页：
      1. 外部链接：直接写网址即可（网址需要写完整，https://）
      2. 本地网页：直接写路径
   2. target：设置跳转方式：
      1. target="\_self"：默认值，在当前窗口中进行跳转，原网页不在了
      2. target="\_blank"：在新窗口中进行跳转，原网页还在

iframe 标签 ​

TIP

链接标签（页中页） ​

概念：一个页面悬浮在另一个页面之上

html
<iframe src="https://www.example.com" width="600" height="400"></iframe>
<!-- 常用属性： -->
<!-- src：指定嵌入内容的 URL 地址-->
<!-- width/height：设置 iframe 的宽度和高度，单位可以是像素或百分比 -->
<!-- title： 提供对 iframe 内容的简要描述，提升可访问性 -->
<!-- frameborder：控制边框显示。frameborder="0" 隐藏边框 -->
<!-- allowfullscreen：允许全屏显示 -->
<!-- sandbox：限制 iframe 内容的权限，比如禁止脚本、表单、插件等，提升安全性 -->
