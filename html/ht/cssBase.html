1. CSS 的介绍 ​

TIP

  1. 定义：层叠样式表
  2. 作用：给页面中的 html 标签设置样式，让页面变得好看

基础样式选择器 ​

1. ID 选择器 （示例） ​

TIP

  1. 结构：#id { css: value }
  2. 作用：找到页面中带有该 id 属性值的标签，设置样式

注意：

  1. id 属性值相当于是身份证号码，在一个页面中是唯一的，不可重复的；
  2. 一个标签上只能有一个 id 属性值；
  3. 一个 id 选择器只能选中一个标签

 

2. 类选择器 （示例） ​

TIP

  1. 结构：.类名 { css: value }
  2. 作用：找到页面中所有带有该类名标签，设置样式
  3. 命名规则：
    1. 由数字、字母、下划线、中划线组成，但是不能以数字或者中划线开头
    2. 命名尽量需要有意义

注意：

  1. 所有的标签都有 class 属性，class 属性的属性值就叫做类名；
  2. 一个标签可以有多个类名，类名之间通过空格隔开；
  3. 一个类选择器可以同时选中多个标签

 

类与 id 的区别：

  1. class 类名相当于姓名，可以重复，一个标签可以同时有多个 class 类名；
  2. id 属性值相当于身份证号码，不可重复，一个标签只能有一个 id 属性值；

实际开发的情况：

  1. 类选择器用的最多；
  2. id 一般配合 js 使用，除非特殊情况，否则不要使用 id 设置样式；
  3. 实际开发中会遇到冗余代码的抽取问题，可以将一些公共的代码抽取到一个公共的类中去

 

3. 标签选择器 ​

TIP

  1. 结构：标签名 { css: value }
  2. 作用：通过标签名找到页面中所有这一类标签，设置样式

注意：

  1. 找的是一类标签，不能单独找到某一个；
  2. 标签选择器无论嵌套关系有多深，都能找到对应的标签

 

4. 属性选择器 （示例） ​

TIP

  1. 结构：元素[属性名] { css: value }
  2. 作用：根据元素的属性值进行选择
  3. 示例：
  1. a[title] { }：存在 title 属性的 <a> 元素
  2. a[href*="example"] { }：存在 href 属性并且属性值包含"example"的 <a> 元素
  3. a[class~="logo"] { }：存在 class 属性并且属性值包含单词"logo"的 <a> 元素

 

5. 通配符选择器 （示例） ​

TIP

  1. 结构：* { css: value }
  2. 作用：找到页面中所有的标签，设置样式

注意：

  1. 针对于大型的项目，一般不会使用通配符选择器，可能会比较消耗浏览器的性能；
  2. 之后可能在基础班的小页面中去除标签的默认 margin 和 padding

 

进阶选择器 ​

1. 复合选择器 ​

TIP

  1. 后代选择器
  1. 结构：选择器1 后代选择器{ css: value }
  2. 作用：先通过选择器 1 找到父标签，然后去找父标签内的后代标签的选择器

 

  1. 子代选择器
  1. 结构：选择器1>子代选择器 { css: value }
  2. 作用：先通过选择器 1 找到父标签，然后去找父标签内的子代标签的选择器

 

  1. 相邻元素的的后面一个选择器
  1. 结构：选择器1+选择器1的下一个兄弟元素{ css: value }
  2. 作用：先通过选择器 1 找到选择器 1 标签，然后去找该元素的后面一个兄弟元素

 

  1. 相邻元素的的后面所有选择器
  1. 结构：选择器1~选择器1的下一个兄弟元素{ css: value }
  2. 作用：先通过选择器 1 找到选择器 1 标签，然后去找该元素的后面所有兄弟元素

 

2. 并集选择器 ​

TIP

  1. 结构：选择器1,选择器2 { css: value }
  2. 作用：找到选择器 1 和选择器 2 对应的标签，设置样式

注意：

  • 并集选择器中可以同时写多个选择器，多种选择器，选择器之间以逗号隔开

 

3. 交集选择器 ​

TIP

  1. 结构：选择器1选择器2 { css: value }
  2. 作用：找到页面中既满足选择器 1 又满足选择器 2 的标签，设置样式

注意：

  1. 交集选择器中可以同时写多个选择器，多种选择器，选择器之间紧挨着的
  2. 如果交集选择器中有标签选择器，此时标签选择器必须写在第一个

 

4. 伪类选择器 ​

TIP

  1. 链接伪类选择器
  1. E:link：a ：标签未访问过的状态
  2. E:visited：a：标签访问过之后的转态
  3. E:hover：鼠标悬停 a 标签的状态
  4. E:active：鼠标按下的状态

注意点：

  1. 书写顺序：lvha
  2. hover 用的最多，并且可以用在其他标签上

 

  1. 焦点伪类选择器：E:focus

 

结构伪类选择器 ​

TIP

  1. first-child
  1. 结构：E:first-child { css: value }
  2. 作用：匹配父元素中的第一个子元素 E

 

  1. last-child
  1. 结构：E:last-child { css: value }
  2. 作用：匹配父元素中的最后一个子元素 E

 

  1. nth-child
  1. 结构：E:last-child { css: value }

伪元素 ​

伪元素

  1. 概念：一般页面中的非主体内容中可以使用伪元素
  2. 区别：
  1. 元素：html 设置的标签
  2. 伪元素：由 css 模拟出来的标签效果
  1. 种类
  1. ::before：在父元素内容的最前面添加一个伪元素
  2. ::after：在父元素内容的最后添加一个伪元素
  1. 注意点：
  1. 有一个必加的属性：content："伪元素中的内容"；
  2. 默认是行内元素

 

CSS 变量 ​

CSS 变量

  1. 使用：
  1. 定义一个变量：--变量名: 值；
  2. 自己及后代元素使用：css属性：var(--变量名)
  1. 注意：
  1. 变量可以被后代元素继承；
  2. 子代的变量会覆盖父级的变量；
  3. :root 代表的是 html 根标签，所有的元素都可以继承它的变量

 

CSS 三大特性 ​

TIP

1. 层叠性 ​

  • 概念：当多个样式作用于同一个元素时，哪个生效由“层叠规则”决定，优先级高的会覆盖低的

 

2. 继承性 ​

  1. 概念：子元素有默认继承父元素样式的效果
  2. 常见的样式可以继承：文字相关属性一般能够被继承
  3. 继承的应用：
  1. 去除列表默认的样式（小圆点）：ul { list-style：none }；
  2. 统一不同浏览器默认文字大小：不同浏览器可能显示的默认字体大小不一致，开发的时候通常会给 body 标签设置统一的 font-size 保证不同浏览器效果一致

 

3. 优先级 ​

  1. 比较重要性：
  1. 作者（开发者）样式表中的!important
  1. !important 写在属性值的后面，分号的前面；
  2. !important 不能提升继承的优先级，只要是继承，优先级最低

  1. 作者样式表中的普通样式

  2. 浏览器默认样式

  1. 比较权重：继承 < 通配符选择器 < 标签选择器 < 类选择器 < id 选择器 < 行内样式 < !important
  1. 选择器的数量：（0,0,0,0）；
  2. 比较选择器，从左往右比较
  1. 行内样式的数量；
  2. id 选择器的数量；
  3. 类选择器、属性选择器、伪类选择器的数量；
  4. 元素选择器、伪元素选择器的数量；

注意点：如果 !important 不是继承，优先级最高

  1. 比较的规则
  1. 从左往右一个一个的比较，如果某一位比较出来了，此时后面的可以不看；
  2. 如果发现比较到最后权重是相同的，此时比较层叠性
  1. 比较代码位置：只有当优先级相同时，此时才会比较代码书写位置
  1. 给同一个标签设置了相同的样式，此时样式会覆盖，写在下面的会生效；
  2. 给同一个标签设置了不同的样式，此时样式会叠加，会共同作用在标签上；

 

CSS 属性值的计算过程 ​

TIP

  1. 背景：渲染出元素时，该元素的所有 css 属性都会有值（默认值也算值）
  2. 定义：一个元素的所有属性值，从没有值到计算赋值的过程
  3. 计算过程（四步）：
  1. 优先声明值：使用样式表（作者、浏览器默认样式表）中没有冲突的声明值，直接作为 css 属性值；
  2. 层叠冲突：对样式表有冲突的声明使用层叠规则，确定 css 属性值；
  3. 使用继承：对仍没有值的属性，如果可以继承，则继承父元素的属性值；
  4. 使用默认值：对仍没有值的属性，则使用默认值；
css
/* 作者样式表 */
.red {
  color: red;        
  font-size: 40px;
}

h1 {
  font-size: 26px;
}

div h1.red {
  font-size: 3em;
  font-size: 30px;
}

/* 浏览器默认样式表 */
h1 {
  display: block;   
  font-size: 2em;
  font-weight: bold;   
}

/* 第一步结果 */
{
  color: red;          
  display: block;      
  font-weight: bold;   
}
