Interface ​

概念 ​

TIP

接口是前后端通信的桥梁

  • 简单理解：一个接口就是服务中的一个路由规则 ，根据请求响应结果

 

接口的组成 ​

TIP

  1. 接口地址（URL）
  2. 请求方法
  3. 请求参数
  4. 响应结果

 

RESTful API ​

TIP

  1. 概念：RESTful API 是一种特殊风格的接口
  2. 主要特点：
  1. URL中的路径表示资源，路径中不能有动词；
  2. 操作资源要与 HTTP 请求方法对应；
  3. 操作结果要与 HTTP 响应状态码对应

 

json-server ​

TIP

  1. 概念：本身是一个 JS 编写的工具包，可以快速搭建 RESTful API 服务
  2. 使用：
  1. 安装：npm i -g json-server；
  2. 创建 JSON 文件，编写基本结构；
  3. 以 JSON 文件所在文件夹作为工作目录 ，执行json-server --watch db.json
