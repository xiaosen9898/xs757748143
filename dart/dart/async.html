Dart的异步操作 ​

TIP

  1. 异步操作的基础：Future

Future的三种状态：

  1. 未完成（pending）：异步操作还没有完成
  2. 完成（completed）：异步操作成功完成，并返回一个结果
  3. 失败（failed）：异步操作失败，抛出了异常
  1. async 和 await 的使用
  1. async：将一个函数标记为异步函数
  2. await：用于等待一个 Future 完成，并获取其结果（await 只能在 async 函数内部使用）
  1. 基本使用：
dart
import 'dart:async';

// 模拟一个异步操作（例如网络请求）
// fetchData 是一个异步函数，它返回一个 Future<String>，代表一个将来会返回字符串结果的异步操作
Future<String> fetchData() async { 
  // 模拟网络延迟，等待 2 秒  
  // Future.delayed 是 Dart 中 Future 类的一个静态方法，用于创建一个延迟一定时间后完成的 Future
  await Future.delayed(Duration(seconds: 2));
  return 'Data fetched';
}

// 异步函数调用
void main() async {
  print('Fetching data...');
  String result = await fetchData();  // 使用 await 等待 fetchData 完成
  print(result);  // 输出：Data fetched
}

  1. 异步错误处理：try-catch
dart
Future<String> fetchDataWithError() async {
  // 模拟一个错误
  await Future.delayed(Duration(seconds: 2));
  throw 'Failed to fetch data';
}

void main() async {
  print('Fetching data...');
  try {
    String result = await fetchDataWithError();  // 如果出错会进入 catch
    print(result);
  } catch (e) {
    print('Error: $e');  // 输出：Error: Failed to fetch data
  }
}

  1. Future 的其他方法
  1. then()：在 Future 完成后执行某个回调，类似于传统的回调函数
  2. catchError()：用于捕获并处理 Future 执行过程中发生的错误
dart
Future<String> fetchDataWithThen() {
  return Future.delayed(Duration(seconds: 2), () => 'Data fetched');
}

void main() {
  fetchDataWithThen()
      .then((result) {
        print(result);  // 输出：Data fetched
      })
      .catchError((e) {
        print('Error: $e');
      });
}
