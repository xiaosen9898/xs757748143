Linux 命令 ​

目录与文件命令 ​

1. 打印当前目录 ​

命令说明

2. 查看目录 ​

命令说明
ll 或者 ls -l列表查看当前目录下的所有目录和文件（列表查看，显示更多信息）

5. 创建文件 ​

命令说明

6. 删除文件或目录 ​

命令说明
删除文件
rm filename删除文件，它不会将文件移到回收站，而是直接删除
rm f1 f2 f3删除多个文件
rm -f filename强制删除文件（不询问确认）
删除目录
rm -r dir_name删除目录及其内容，如果目录中有文件或子目录，rm 需要使用 -r（递归）选项来删除目录及其内容
rm -rf dir_name强制删除目录及其内容，使用 -rf 选项，可以在没有任何确认的情况下强制递归删除目录及其所有内容
rm -rf /path/to/destination/强制删除/path/to下的/destination/目录及其内容
rmdir empty_dir当目录为空时，才能被删除；如果你不知道目录是否为空，可以使用 rm -r 删除目录
sudo rm -rf /*这个命令会删除从根目录开始的所有文件和目录，包括系统文件、程序文件、配置文件、用户文件等。由于使用了 sudo，它会以 root 用户的身份执行，因此没有权限限制，会删除系统中的一切内容

 

7. 移动文件或目录 ​

命令说明
文件
mv file.txt /path/to/destination/将 file.txt 文件从当前目录移动到 /path/to/destination/ 目录
mv file1.txt file2.txt /path/to/destination/将 file1.txt 和 file2.txt 文件移动到指定的目标目录
mv oldname.txt newname.txt这个命令将 oldname.txt 文件重命名为 newname.txt
mv -i file.txt /path/to/destination/如果目标目录中已存在同名文件，-i 选项会提示你是否覆盖文件
mv -n file.txt /path/to/destination/-n 选项用于禁止覆盖任何已存在的文件，即使目标位置已有同名文件，mv 也不会覆盖它
mv -f file.txt /path/to/destination/使用 -f 选项强制覆盖，不会给出确认提示
mv oldname.txt /path/to/destination/newname.txt将文件 oldname.txt 移动到目标目录 /path/to/destination/，并将其重命名为 newname.txt
目录
mv dir1/ /path/to/destination/将目录 dir1/ 从当前目录移动到 /path/to/destination/
mv olddir/ newdir/这个命令将目录 olddir/ 重命名为 newdir/

 

8. 查找目录或文件 ​

命令说明
文件
find . -name "file.txt"查找当前目录及子目录下名为 file.txt 的文件
find . -iname "file.txt"忽略大小写查找文件
find . -type f查找当前目录及其子目录下的所有文件（不包括目录）；可以将 . 替换成具体路径表示特定目录下查找
find . -size +10M查找大于 10MB 的文件
find . -size -100k查找小于 100KB 的文件
find . -mtime -7查找最近 7 天内修改过的文件
find . -mtime +30查找 30 天前修改的文件
find . -name "*.log" -exec rm -f {} \;查找并删除所有 .log 文件
find . -name "*.sh" -exec chmod +x {} \;查找并更改文件权限（例如，给所有 .sh 文件加执行权限）
find . -name "*.txt"只显示匹配的文件路径（默认行为）
find . -name "*.txt" -exec ls -l {} \;查找并显示详细信息（类似 ls -l）
find . -name "*.txt" -size +1M -mtime -10查找大于 1MB 且在最近 10天内修改的 .txt 文件
find . \( -name "*.txt" -o -name "*.log" \)查找多个条件（例如同时匹配 .txt 和 .log 文件）
目录
find . -type d查找所有目录

 

9. 复制文件和目录 ​

命令说明
cp [选项] 源路径 目标路径
文件
cp file.txt /path/to/destination/将文件 file.txt 复制到 /path/to/destination/ 目录
cp file1.txt file2.txt /path/to/destination/将 file1.txt 和 file2.txt 复制到 /path/to/destination/
cp file.txt /path/to/destination/newfile.txt将 file.txt 复制到 /path/to/destination/ 并重命名为 newfile.txt
cp -f file.txt /path/to/destination/如果目标文件已经存在，cp 默认会覆盖它。如果不希望被提示，使用 -f（force）选项强制复制并覆盖目标文件
cp -p file.txt /path/to/destination/使用 -p 选项可以保留文件的时间戳、权限、所有者等属性
cp -i file.txt /path/to/destination/使用 -i（interactive）选项，在目标文件已存在时，cp 会提示用户确认是否覆盖
cp -P symlink /path/to/destination/如果要复制符号链接（symlink），而不是复制符号链接指向的文件，可以使用 -P 选项
目录
cp -r dir1/ /path/to/destination/将目录 dir1/ 复制到 /path/to/destination/

cat命令 ​

命令说明
cat [选项] 文件...
用于查看文件内容、合并文件内容、以及创建文件的常用 Linux 命令
cat file.txt查看 file.txt 文件的内容
cat file1.txt file2.txt一次性查看多个文件的内容，cat 会按顺序显示所有文件的内容
cat file1.txt file2.txt > combined.txt将 file1.txt 和 file2.txt 的内容合并，并保存到 combined.txt 文件中。如果 combined.txt 文件已经存在，它会被覆盖
cat file1.txt >> file2.txt将 file1.txt 的内容追加到 file2.txt 文件的末尾，而不是覆盖它
cat -n file.txt输出 file.txt 的内容，并为每一行显示行号
cat -E file.txt在每行的末尾显示一个 $ 符号
cat -T file.txt显示文件中的制表符字符（如果有的话），并将它们显示为 ^I
cat -v file.txt它会以可见的方式显示非打印字符
cat > newfile.txt让你输入文本，当你按下 Ctrl+D 结束输入时，内容会被写入到 newfile.txt 文件中
cat -s file1.txt file2.txt有时候你可能希望在多个文件之间添加分隔符。可以使用 -s 选项来在连续的空行之间压缩为一行

 

wc命令 ​

命令说明
wc [选项] [文件名]
用于统计文件中字符数、单词数和行数的命令
wc -l example.txt只统计行数；输出：10 example.txt
-l只显示行数（lines）
-w只显示单词数（words）
-c只显示字节数（bytes）
-m只显示字符数（characters）。注意，这与 -c 不同，-m 统计的是字符数量，支持多字节字符
-L显示最长行的长度（longest line length）

命令说明
grep [选项] 模式 [文件...]
用于在文件中搜索指定的模式（字符串或正则表达式）
grep "hello" example.txt输出文件 example.txt 中包含 hello 的行
grep -i "hello" example.txt忽略大小写搜索

chmod命令 ​

命令说明
chmod [选项] 模式 文件名
用于更改文件或目录的权限；权限分为三种：读（r）、写（w） 和 执行（x），分别针对三类用户：所有者（owner）、组（group） 和 其他人（others）
1. 权限的数字表示法
读 (r) = 4；
写 (w) = 2；
执行 (x) = 1；
无权限 (-) = 0		文件权限由 3 位八进制数字组成，每位分别表示所有者、组和其他人的权限，每种权限用数字表示
chmod 755 file所有者：rw-，组：r--，其他人：r--
chmod 644 file.txt所有者有 rwx，组和其他人有 r-x
2. 符号表示法
u：所有者（user）；
g：组（group）；
o：其他人（others）；
a：所有用户（all，等同于 ugo）		用户类别
chmod u+x file为所有者添加执行权限
chmod g-w file移除组的写权限
chmod o=r file设置其他人为只读权限
chmod -R 755 /path/to/directory递归修改目录权限
chmod --reference=reference.txt file.txt设置与参考文件一致的权限
chmod o= file.txt移除其他人的所有权限

 

用户管理 ​

1. 创建用户 ​

useradd [选项] 用户名
命令说明
-m为用户创建主目录（通常是 /home/用户名）
-d [目录]指定主目录路径
-s [shell]指定用户登录后使用的默认 shell（如 /bin/bash）
-g [组名]指定用户的初始组
-G [组名]指定用户的附加组
-u [UID]指定用户的 UID
-c [描述]添加用户描述信息
useradd -m -s /bin/bash -c "开发人员" devuser

 

2. 修改用户 ​

usermod [选项] 用户名
命令说明
-l [新用户名]修改用户名
-d [新主目录] -m移动并修改主目录
-s [shell]修改默认 shell
-g [组名]修改初始组
-G [组名1,组名2]修改附加组
-L锁定用户账户
-U解锁用户账户
usermod -s /bin/zsh devuser

 

3. 删除用户 ​

userdel [选项] 用户名
命令说明
-r 用户名删除用户的同时，删除其主目录及相关文件
userdel -r devuser

 

4. 查看用户信息 ​

命令说明
whoami查看当前用户
who查看登录用户
cat /etc/passwd查看所有系统用户

 

5. 设置用户密码 ​

命令说明
passwd当前用户修改密码
passwd devuser管理员为其他用户设置密码

 

6. 用户组管理 ​

命令说明
1. 创建用户组
groupadd [组名]创建用户组
2. 修改用户组
groupmod -n [新组名] [旧组名]修改用户组
3. 删除用户组
groupdel [组名]删除用户组
4. 将用户添加到组
usermod -aG [组名] [用户名]将用户添加到组

 

7. 切换用户 ​

命令说明
su [用户名]切换到另一个用户
sudo [命令]需要当前用户在 sudoers 文件中具有权限

 

系统/进程命令 ​

1. kill ​

命令说明
kill [选项] [PID]
通常用来终止进程
kill [PID]终止进程
kill -9 [PID]强制终止进程；发送 SIGKILL 信号，立即终止目标进程
kill -s HUP [PID]发送 SIGHUP 信号，通常用于重新加载服务配置
kill -9 [PID1] [PID2] [PID3]终止多个进程
kill -18 [PID]继续运行已暂停的进程
kill -l列出所有信号
ps aux | grep [进程名]
kill [PID]		使用 ps 或 pgrep 查找进程后用 kill 终止
pkill [进程名]直接通过进程名终止
kill -9 $(pgrep [进程名])结合 pgrep 和 kill，通过进程名终止
kill -u [用户名]终止某用户的所有进程
权限问题：1.普通用户只能终止自己拥有的进程；2.需要使用 sudo 终止其他用户的进程或系统级进程

ps aux查看所有进程

ps -u root查看特定用户的进程
ps aux | grep apache结合 grep 查找特定进程

 

3. free ​

5. vmstat ​

6. iostat ​

iostat -p sda显示指定设备的统计信息

history命令 ​

history [选项] [n]
用于显示和管理 Linux 命令历史的命令。它可以帮助你查看之前执行过的命令、重复执行历史命令以及清理历史记录等
history -c清空所有历史记录
history 10查看最后 10 条历史记录
history -d 10删除历史记录中的第 10 条命令

history -n读取未添加到历史文件中的命令
!10执行编号为 10 的命令
!!执行最后一条历史命令
!ls查找并执行最近一个以 ls 开头的命令
cat ~/.bash_history查看历史文件中的命令

时间日期类 ​

命令说明
date [选项] [+格式]
用于显示和设置系统日期和时间的 Linux 命令。它可以显示当前的日期和时间，并支持多种格式化选项来修改输出的日期和时间格式
date -u查看当前 UTC 时间
date "+%Y-%m-%d %H:%M:%S"自定义日期和时间格式
date -d "2024-12-25 12:00:00"显示指定日期和时间
date -d "next Monday"显示下一个星期一的日期
sudo date --set="2024-12-25 12:00:00"将当前时间设置为 2024 年 12 月 25 日 12:00:00
date -R显示 RFC 2822 格式的日期
date -I显示 ISO 8601 格式的日期
date -r myfile.txt查看文件的最后修改时间
