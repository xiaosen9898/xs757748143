核心概念 ​

TIP

1. State ​

  1. 作用：存储数据

  2. 注意：Redux不负责驱动视图，只是负责计算数据、保存数据

 

2. Action ​

  1. 作用：描述一个动作

  2. 本质：具体type字段的JS对象

  3. 传参：payload属性用来携带参数（不建议自定义payload的名字）

注意：

  • 推荐使用函数的返回对象的方式
  • Action默认只能处理同步行为，不能处理异步行为

 

3. Dispatch ​

  1. 作用：触发一个action

  2. 特点：redux中修改state唯一的方式就是dispatch(Action)

 

4. reducer ​

  1. 作用：负责计算新的状态

  2. 本质：纯函数

  3. 语法：(state, action) => newState

  4. 推荐写法：

  1. 单独的Reducer函数
  2. 单独的变量维护state的初始值
  3. switch语句替换if语句
  1. 注意：
  • 不可变数据，新值覆盖旧值
  • 是纯函数，不能写请求、操作DOM

 

5. Subscriber ​

  • 作用：组件可以订阅 store 中的状态变化，以便更新 UI

 

Redux 代码执行过程 ​

TIP

  1. 创建 store 时，Redux 就会先调用一次 reducer，来获取到默认状态
  2. 当你需要更新状态时，就先分发动作 store.dispatch(action)
  3. Redux 内部，store 就会调用 reducer，传入：上一次的状态（当前示例中就是：10）和 action（{ type: 'increment' }），计算出新的状态，并返回
  4. reducer 执行完毕后，将最新的状态交给 store，store 用最新的状态替换旧状态，状态更新完毕

 

Redux基本使用 ​

TIP

  1. 准备仓库 src\store\index.js
js
import { createStore } from 'redux'

// state 初始值
const initState = {
  msg: '',
  count: 0,
}

// reducer 函数
function reducer(state = initState, action) {
  console.warn('Redux Logger', { action, oldState: state })
  switch (action.type) {
    case 'add':
      return { ...state, count: state.count + 1 }
    case 'minus':
      return { ...state, count: state.count - 1 }
    default:
      return state
  }
}

// 创建仓库实例
const store = createStore(reducer)

// 导出仓库实例
export default store

import { useEffect, useState } from 'react'
import store from './store'

function Son() {
  const [count, setCount] = useState(0)                        
  const [count, setCount] = useState(store.getState().count)   

  useEffect(() => {                                            
    // 组件加载订阅
    console.log('组件加载订阅')                                 
    const unSubscribe = store.subscribe(function () {          
      setCount(store.getState().count)                         
    })                                                         
    // 组件卸载取消订阅
    return () => {                                             
      console.log('组件卸载取消订阅')                            
      unSubscribe()                                             
    }                                                           
  }, [])                                                        
 return (
   <div>
     <h2>子组件</h2>
     <h2>count:{count}</h2>
      <button onClick={() => setCount(count + 1)}>+1</button>
      <button onClick={() => store.dispatch({ type: 'add' })}>+1</button>
      <button onClick={() => setCount(count - 1)}>-1</button>
      <button onClick={() => store.dispatch({ type: 'minus' })}>-1</button>
    </div>
  )
}
