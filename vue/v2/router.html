单页面应用程序 ​

TIP

  1. 定义：跳转页面，不会刷新，客户端切换内容（SPA（single page application））
  2. 优点：用户体验比较好
  3. 缺点：
  1. 首次加载往往比较慢（动态加载）
  2. 不利于 SEO（解决：服务端渲染）

配置路由 ​

TIP

  1. 创建路由代码文件：src/router/index.js
  2. 配置路由：
js
/* src/router/index.js */
import Vue from 'vue';
import VueRouter from 'vue-router';

// 1.导入组件
import Home from '../views/Home.vue';
import About from '../views/About.vue';

// 2.使用 VueRouter 插件
Vue.use(VueRouter);

// 3.定义路由规则
const routes = [
  {
    path: '/',        // 路由路径
    name: 'Home',     // 路由名称
    component: Home   // 关联的组件
  },
  {
    path: '/about',
    name: 'About',
    component: () => import('../views/About.vue')  // 按需加载组件
  }
];

// 4.创建路由实例
const router = new VueRouter({
  mode: 'history',    // 使用 history 模式，去除 URL 中的 # 号
  routes              // 注入路由
});

// 5.导出路由实例
export default router;

路由的模式 ​

TIP

  1. 两种路由模式：
  1. hash 路由：浏览器地址栏中会显示#，如：http://localhost:8080/#/login
  2. history 路由：浏览器地址栏中不显示#，如：http://localhost:8080/login
  1. 模式设置：
js
const  router  =  new  VueRouter( {
  routes,
  mode: "history"
} )

重定向和 404 ​

TIP

1. 重定向： ​

  1. 访问一个页面时强制转到另一个页面
  2. 路由配置：{path: '/', redirect: '要转到的页面路径' }

 

2. 404 页面 ​

  1. 访问不到页面时跳转到的页面
  2. 在路由最后一项配置：{ path：'*', component: NotFound}

 

导航方式 ​

TIP

1. 声明式导航 ​

  1. 语法：<router-link>
  2. 特点：
  1. 替代 a 标签（实质渲染为 a 标签），能跳转会高亮（自带激活时的类名）；
  2. to 属性等价于提供 href 属性；
  3. 类名router-link-active：激活的导航链接（模糊匹配）；
  4. 类名router-link-exact-active：激活的导航链接（精确匹配）；

 

2. 编程式导航 ​

  1. 语法：this.$router.push()
  2. 特点：
  1. 用方法跳转：this.$router.push("页面路径")；
  2. 返回上一页：this.$router.back()

 

  • 两种导航的区别：
  1. 声明式导航需要用户触发；
  2. 声明式导航是 a 变迁，可以复制链接分享；
  3. 编程式导航比较灵活

 

导航守卫 ​

TIP

  1. 作用：
  1. 检测没有 token 的页面；
  2. 处理一些全局的事件（获取用户信息、权限等）

 

  1. 全局前置守卫：

三个参数：

  1. to：即将要进入的目标路由对象；
  2. from：当前导航正要离开的路由；
  3. next：去某个页面
查看
js
// permission.js

// 假设我们有一个函数用于检查用户是否已登录
function isLoggedIn() {
  return !!localStorage.getItem('user');
}

// 全局前置守卫
router.beforeEach((to, from, next) => {
  if (to.matched.some(record => record.meta.requiresAuth)) {
    // 如果需要登录且未登录，跳转到登录页面
    if (!isLoggedIn()) {
      next({ name: 'Login' });
    } else {
      next();
    }
  } else {
    next(); // 确保继续导航
  }
});
