SQL ​

通用语法 ​

TIP

  1. SQL语句可以单行或多行书写，以分号结尾

  2. MySQL的SQL语句不区分大小写，建议关键字大写

  3. 注释：

  1. -- 注释内容：单行注释
  2. # 注释内容： MySQL特有
  3. /* 注释内容 */：多行注释

 

SQL分类 ​

TIP

1. DDL ​

  • 数据定义语言，用来定义数据库对象（数据库、表、字段）

 

2. DQL ​

  • 数据查询语言，用来查询数据库中表的记录

 

3. DML ​

  • 数据操作语言，用来对数据库表中的数据进行增删改

 

4. DCL ​

  • 数据控制语言，用来创建数据库用户、控制数据库的访问权限

 

DDL - 定义数据库对象 ​

TIP

1. 数据库操作 ​

  1. 查询：
sql
查询所有数据库：show databases;
查询当前数据库：select database();

 

  1. 创建：
sql
创建数据库：create database 数据库名;

 

  1. 删除：
sql
删除数据库：drop database 数据库名;

 

  1. 切换：
sql
切换到数据库：use 数据库名;

 

2. 表操作 ​

  1. 查询：
sql
查询当前数据库所有表：show tables;
查询表结构：desc 表名;
查询指定表的建表语句：show create table 表名;

 

  1. 创建：create database 数据库名;
sql
create table 表名(
   字段1 字段1类型[ comment '字段1注释' ]
   字段2 字段2类型[ comment '字段2注释' ]
)  [ comment '表注释' ];
sql
create table emp(
   id int comment '编号',
   workno varchar(10) comment '工号',
   name varchar(10) comment '姓名', 
   gender char(1) comment '性别', 
   age tinyint unsigned comment '年龄',
   idcard char(18) comment '身份证号',
   entrydate date comment '入职时间'
) comment '员工表';

  1. 修改：
sql
修改表名：alter table 表名 rename to 新表名;

 

  1. 删除：

删除表：drop table 表名;
删除并重新创建：truncate table 表名;

 

3. 表字段操作 ​

  1. 添加：
sql
添加字段：alter table 表名 add 字段名 类型(长度) [ comment '注释' ] [约束];

 

  1. 修改：
sql

DQL - 查询数据库中表 ​

TIP

总语法 ​

sql
select 字段列表 from 表名 where 条件 group by 分组字段列表 having 分组后条件列表 order by 排序字段列表 limit 分页参数

 

1. 基础查询 ​

  1. 查询所有字段：
sql
select * from 表名;

 

  1. 查询某个/些字段：
sql
select 字段1,字段2,,... from 表名;

 

  1. 查询并且去除重复记录：
sql
select distinct 字段1,字段2,,... from 表名;

 

  1. 查询某个/些字段并设置别名：
sql
select 字段1[as 别名1] ,字段2[as 别名2],,... from 表名;

 

2. 条件查询 ​

  1. 语法：
sql
select 字段1,字段2,,... from 表名 where 条件列表;
  1. 条件：
  1. 比较运算符：> 、>= 、!= 、in(...) 、like 占位符 、is null
  2. 逻辑运算符：and && 、or || 、not !
  3. like的用法：
  1. 查询某个字段为两个字符的：
sql
select 字段列表 from 表名 where 字段 like '__';
  1. 查询某个字段最后一个字符为x的：
sql
select 字段列表 from 表名 where 字段 like '%x';

 

3. 聚合函数 ​

  1. 语法：
sql
select 聚合函数(字段列表) from 表名 [where 条件];
  1. 作用：将一列作为一个整体进行计算
  2. 常见聚合函数：
  1. 统计数量：count()
  2. 最大/最小值：max()/min()
  3. 平均值：avg()
  4. 求和：sum()
  • 注意：null不参与运算

 

4. 分组查询 ​

  1. 语法：
sql
select 字段列表 from 表名 [ where 条件] group by 分组字段名 [having 分组后过滤条件];
  1. 执行顺序：where > 聚合函数 > having

 

5. 排序查询 ​

  1. 语法：
sql
select 字段列表 from 表名 order by 字段1 排序方式1, 字段2 排序方式2;
  1. 排序方式：
  1. 升序(默认)：asc
  2. 降序：desc
  • 注意：如果是多字段排序，当第一个字段值相同时，才会根据第二个字段进行排序

 

6. 分页查询 ​

  1. 语法：
sql
select 字段列表 from 表名 limit 起始索引, 查询记录数;
  1. 注意：
  1. 起始索引从0开始，起始索引 = (查询页码 - 1 ) * 每页显示记录数
  2. 分页查询是数据库的方言，不同的数据库有不同的实现，MySQL中是limit
  3. 如果查询的是第一页数据，起始索引可以省略，直接简写为limit 10

 

DML - 表中的数据进行增删改 ​

TIP

1. 插入数据 ​

  1. 指定字段添加数据：
sql
insert into 表名(字段1, 字段2, ...) values(值1, 值2, ...);
  1. 全部字段添加数据：
sql
insert into 表名 values(值1, 值2, ...);
  1. 批量添加数据：
sql
insert into 表名(字段1, 字段2, ...) values(值1, 值2, ...), values(值1, 值2, ...), ...;
insert into 表名 values(值1, 值2, ...), values(值1, 值2, ...),values(值1, 值2, ...);

 

2. 修改数据 ​

  1. 修改整张表所有数据：
sql
update 表名 set 字段名1 = 值1, 字段名2 = 值2, ... ;
  1. 有条件修改数据：
sql
update 表名 set 字段名1 = 值1, 字段名2 = 值2, ... where 条件;

 

3. 删除数据 ​

  1. 删除整张表所有数据：
sql
delete from 表名;
  1. 有条件删除数据：
sql
delete from 表名 where 条件;

 

DCL - 数据库的权限 ​

TIP

1. 用户管理 ​

  1. 查询用户：
sql
use mysql;
select * from user;
  1. 创建用户：
sql
create user '用户名'@'主机名' identified by '密码';
  1. 修改用户密码：
sql
alter user '用户名'@'主机名' identified with mysql_native_password by '新密码';
  1. 删除用户：
sql
drop user '用户名'@'主机名';

 

2. 权限控制 ​

  1. 查询权限：
sql
show grants for '用户名'@'主机名';
  1. 授予权限：
sql
grant 权限列表 on 数据库名.表名 to '用户名'@'主机名';
  1. 撤销权限：
sql
revoke 权限列表 on 数据库.表名 from '用户名'@'主机名';
