小程序云开发 ​

uniapp ​

TIP

  1. 介绍：云开发指的是微信小程序提供了开发项目的完整配套环境
  • 数据库
  • 后台接口
  • 云空间

 

  1. 开通云开发环境：
  1. 创建项目时，后端服务选择 → 小程序·云开发
  2. 新建云环境（一个项目对应一个环境）
  • 在开发者工具中点击云开发
  • 创建新环境（默认可拥有最多两个环境。在实际开发中，建议每一个正式环境都搭配一个测试环境）
  1. 云开发官方文档资料

 

  1. 操作数据库：

  1. 添加表(集合)

  2. 添加字段(数据)

  3. 设置表格的权限

  4. 小程序代码中选择环境

  5. 暂时删除无用文件

  6. 应用启动时 初始化 云开发环境

  7. 查询数据库

js
//app.js
App({
  onLaunch: function (options) {
    // 1. 初始化 云开发环境
    wx.cloud.init({
      // 2.1 test 为云开发的环境 id

      env: "cloud-fun-3w6ja",
    });

    // 2. 创建数据库对象
    const db = wx.cloud.database({});

    // 3. 创建集合对象
    const todos = db.collection("todo");
    // 4. 查询数据
    todos.get().then(console.log);
  },
});

  1. 操作云存储：

  1. 说明：云存储让我们在小程序中可以直接上传文件到自己的环境中,也可以下载和删除

  2. 使用：

js
<image bindtap="chooseImage" mode="widthFix" src="{{src}}"></image>
1
js
Page({
  data: {
    src: "https://s1.ax1x.com/2020/08/16/dVJwRg.png",
  },
  // 选择图片
  chooseImage() {
    wx.chooseImage({
      sizeType: ["original", "compressed"],
      sourceType: ["album", "camera"],
      success: (result) => {
        // 获取选中的图片路径
        // http://tmp/wxad258ddc70c8662c.o6zAJsxXHR0g6hvH4eLnOTABv8pQ.GQFTk88TDq8P70746eb521c1ff2e957ff92a8f046855.png
        const tempFilePath = result.tempFilePaths[0];

        const index = tempFilePath.lastIndexOf(".");
        // 获取文件的后缀名
        const extName = tempFilePath.substr(index); //  .png
        // 随机生成新的名称
        const fileName = parseInt(Math.random() * Date.now()) + extName;

        // 上传到云存储
        wx.cloud.uploadFile({
          cloudPath: fileName, // 上传至云端的路径
          filePath: tempFilePath, // 小程序临时文件路径
          success: (res) => {
            this.setData({
              src: res.fileID, // 上传后的图片路径
            });
          },
          fail: console.error,
        });
      },
      fail: () => {},
      complete: () => {},
    });
  },
});

  1. 定义：条件编译是uniapp中的一种根据特殊的注释来实现不同平台下运行不同代码的功能的技术

  2. 使用：

  1. 语法：

   需条件编译的代码
#endif
